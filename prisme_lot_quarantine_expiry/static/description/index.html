Description and Features

If a batch / lot due date is overdue, the batch lot will be moved to Quality management module (it is like a “quarantine” status) and picking will be created for quality check. If quality agree for new due date, batch / lot is accepted and comeback into inventory otherwise batch / lot must be scrapped.

Module Features :

  • Inventory
    • method to automatically move the batch lot to the Quality management module when the due time is overdue
  • Quality management
    • method to move the batch lot to the inventory again if the quality agree a new due date
    • method to scrap the batch if the quality don't agree a new due date
