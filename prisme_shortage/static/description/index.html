Description and Features

Add functionalities to compute and show a list of shortage products.

Module Features :

  • New Product shortage list view : to display shortage products
    • Group: stock moves are grouped by product variant
    • Origin
    • Partner
    • Product
    • Quantity
    • Forecast
    • Forecast short
    • Source Location
    • Destination location
    • Date
    • State
  • Wharehouse/Traceability
    • new "All product shortage" button to open all products shortage list view
  • Product (only variants)
    • New Shortcut button "Product shortage"
      • to open product shortage list view for selected product.

Installation

  1. Click the install button and it's done
